# Custom Children's Playhouse Website

This is a ready-to-upload static website for advertising custom made-to-order children's playhouses.

## Files
- `index.html` — main website page
- `styles.css` — all styling
- `script.js` — mobile menu and placeholder form behavior

## Customize before launch
Search inside `index.html` and replace:
- `Little Haven Playhouses`
- `(207) 555-1234`
- `hello@yourplayhousebusiness.com`
- Gallery placeholder panels with real photos
- Quote form `action="#"` with your real form endpoint

## Form options
You can connect the form to:
- Formspree
- Netlify Forms
- Squarespace form block
- Shopify contact form
- Your own PHP/Node backend

## Upload options
This site can be hosted on:
- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- Any normal web hosting account
